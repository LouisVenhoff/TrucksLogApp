Beschreibung

Dieses Projekt bietet eine einfache Möglichkeit, TLS/SSL-Zertifikate zu generieren und Anmeldeinformationen für einen Datenbankserver zu konfigurieren, ohne dass eine Benutzeroberfläche geöffnet wird. Die Schritte werden automatisch im Hintergrund ausgeführt, um den Prozess zu vereinfachen.

Anforderungen

Betriebssystem: Windows, Linux, macOS
NodeJS installiert
Installation und Verwendung


Führe die Software aus. Sie wird automatisch gestartet und geschlossen. Dadurch wird der Ordner "certificates" im Hauptverzeichnis erstellt.
Gehe in den "certificates"-Ordner und füge deine Schlüsseldateien  "server.key" und deine Zertifikatsdatei  "server.crt" ein.
Achte auf korrekte benamung.
Konfigurationsdatei erstellen:
Führe die Software erneut aus. Sie wird automatisch gestartet und geschlossen. Dadurch wird die Datei "config.json" im Hauptverzeichnis erstellt.
Öffne die "config.json"-Datei mit einem Texteditor und fülle den vorgefertigten JSON-String mit den Anmeldeinformationen für deinen Datenbankserver aus.

Software starten:
Führe die Software erneut aus. Der Dienst wird gestartet und steht zur Verfügung.
Hinweise

Stelle sicher, dass du die Zertifikatsdateien "server.key" und "server.crt" im "certificates"-Ordner platziert hast, bevor du die Software erneut ausführst.
